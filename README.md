# Python Learning & Exam Performance Dataset

Python final exam score prediction using Linear Regression, Random Forest и XGBoost.

### Conclusions
- Best performing model — **Linear Regression** (R² ≈ 0.79)
- Main factors: prior programming experience, self reported confidence in Python, hours of learning per week, number of completed projects
- Country of a student does not affect final exam score

Notebook includes EDA, modelling and models' performance comparison
